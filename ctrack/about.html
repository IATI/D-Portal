d-portal.org aims to be a simple user interface to search and browse all IATI data relating to a specific country.

The final product will be available as a set of HTML, Javascript, and CSS files.

Since HTML/CSS/Javascript are the technologies of the web, and are widely used and understood, we believe this is a good option for easy adoption by others, for easy deployment, and for easy re-configuration/customisation.

What is it?

d-portal.org is a country-based information platform that tracks resource flows. It is aimed at providing line ministries, parliamentarians and civil society with information that can assist with the planning and monitoring of development activities.

In its first iteration it contains current data published through the International Aid Transparency Initiative (IATI) as well as the most recent (2012) data published by the OECD DAC Creditor Reporting System. Future development will include, inter alia, data from Aid Information Management Systems, national budgets and a range of comparable socio-economic indicators.

In its first iteration the platform poses two questions: where do resources come from, and where do they go (by sector and location)? Recent, current and future data is presented and compared across data sources.

In order to answer these questions from a country perspective it is insufficient merely to display available data: it is equally important to plot the gaps in the information. Coping with poor quality or missing data is an essential part of the information interface.

d-portal.org is an open source platform. While it provides a generic deployment for any country it has been developed with the specific intention of being customised in-country to meet local needs. A prototype local application for Nepal is currently being developed in parallel with the core engine.

d-portal.org has been jointly funded by IATI and Development Initiatives (DI). IATI is keen to promote the access to IATI-formatted data at country level. DI is committed to improving access to information by implementing joined-up data solutions.

SAVi is a simple activity viewer for IATI XML that you can view in real time, online, in your favourite browser.

SAVi uses CSS and javascript to display otherwise complex-looking XML into a format that is easier to read and digest.

dstore subsidizes the IATI Datastore with an optimized nodejs + SQLite version for use in real time queries.
Q allows you to query information you need via simple but complex filters.
Here are some Query examples:
http://d-portal.org/q.xml?limit=10&orderby=day_start&day_start_gt=1950-01-01
Returns XML which is automatically formatted into HTML, source remains pure XML
This query displays activities limiting the amount to 10 limit=10 sorted by starting day orderby=day_start and from 1950 onwards day_start_gt=1950-01-01.
http://dev.ctrack.iatistandard.org/q.xml?from=jml,act,sector,country&country=bd&sector_percent_not_null
Returns XML which is automatically formatted into HTML, source remains pure XML
This query displays all activities within Bangladesh (country code is BD) country=bd that have sectors reported in the data sector_percent_not_null. Ultimately, this query allows you to look at all the activities that have graphs in them.
http://d-portal.org/q?from=act,country&groupby=reporting_org&select=stats&country_code=BD
Returns JSON (JSON is default) data about activites (activites are default) where 100 rows from the database is the default limit
This query displays data statistics select=stats about activites for each reporting organisation groupby=reporting_org for Bangladesh country_code=BD.
http://d-portal.org/q.csv?from=trans&orderby=trans_day&limit=1000
Returns CSV file (named q.csv by default)
This query allows you to download the CSV q.csv of 1000 limit=1000 transaction data from=trans sorted by date orderby=trans_day.
