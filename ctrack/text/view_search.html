#more_search_intro
The new DISCOVER feature helps find activities published to the IATI Standard based on keywords - it was originally developed to help people who wanted to find information on specific humanitarian crises, including the Ebola outbreak.

This is a new feature of d-portal and development is ongoing - as always feedback (email or GitHub) is very welcome.

  • DISCOVER allows you to type in one or more keywords – it searches the Titles and Descriptions of all IATI activities and provides results in the usual d-portal format (stats and charts which you can explore by clicking through to tables)
  • Refine your search by adding multiple words – eg ‘Hurricane’ will find more results than ‘Hurricane Matthew’
  • If you type multiple words, the results will display activities which contain BOTH or ALL of those words (not necessarily in that order)
  • Leaving a space between words is currently the same as using a + symbol. Using quotation marks around a phrase e.g. “World Humanitarian Summit” currently gives the same results as using +. We aim to update the function to allow searching an exact phrase using quotation marks shortly.
  • Adapted for English plural nouns - eg typing ‘Refugee’ will include results for ‘Refugees’
