GLOSSARY
Quick search Quick buttons for specific searches.
The COVID-19 search follows the IATI Publishing Guidance. It currently searches the database for COVID-19 in title, activity and transaction description fields and looks for the relevant codes in the humanitarian-scope and tag element.
Text search Search all narrative fields of any languages within all published IATI activities.
ABC - 123 Sort dropdowns by alphabetical order or by relevant IATI/CRS code (e.g. by CRS sector numbers).
{search_status_txt} An IATI code defining the current status of the activity.
{search_collaboration_type_txt} An IATI code defining the type of collaboration involved in the activityâ€™s disbursements.
{search_aid_type_txt} An IATI code specifying the type of aid being supplied.
{search_finance_type_txt} An IATI code for the default DAC/CRS transaction classification used to distinguish financial instruments like grants or loans.
{search_flow_type_txt} An IATI code defining whether the activity is funded by Official Development Assistance (ODA), Other Official Flows (OOF), etc.
{search_tied_status_txt} An IATI code interpreting the usage of Columns 36-38 of the CRS++ reporting format.
{search_document_category_txt} An IATI code defining the category of the document.
{search_humanitarian_activity_txt} A process flag to indicate that this activity and/or transaction relates entirely or partially to humanitarian aid.
{search_humanitarian_scope_txt} An IATI code for a recognised vocabulary of terms classifying the event or action.
{search_participating_txt} Organisations from IATI Registry - not an extensive list and includes only the existing list of IATI Reporting Organisations.
{search_policy_txt} Uses both IATI code for the vocabulary to be used to define a policy or theme addressed by the activity and OECD DAC CRS code indicating the significance of the policy marker for this activity.
{search_publisher_txt} Organisations are from the IATI Registry. This IATI element is a list of reporting organisations issuing the report.
{search_recipient_txt} Countries reported by publishers are using ISO3166 Alpha-2.
{search_related_activity_txt} An IATI code for the type of relationship another separately reported IATI activity is related to this one.
{search_result_type_txt} An IATI code for the type of result being reported.
{search_sector_txt} The sector reported corresponds to an OECD DAC CRS 5-digit purpose code.
{search_sectorgroup_txt} The sector reported corresponds to an OECD DAC CRS 3-digit purpose code.
{search_tag_vocab_txt} An IATI code for the vocabulary or taxonomy used for tag classifications.
{search_commitment_txt} Select the minimum and/or maximum sum of all Outgoing Commitment type transactions.
{search_transaction_type_txt} An IATI code defining the type of transaction.
{search_year_txt} Select the starting and/or ending year as the range for activities. This looks at actual and planned dates for activities that take place during the min and max year; ie. activities that started before the end of the max year and ended after the start of the min year to get an overlap of the ranges.
